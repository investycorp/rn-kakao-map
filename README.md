# rn-kakao-map

Kakao Maps SDK for React Native (iOS, Android)

## Installation

```sh
npm install rn-kakao-map
```

## Usage

```js
import { RnKakaoMapView } from "rn-kakao-map";

// ...

<RnKakaoMapView color="tomato" />
```
